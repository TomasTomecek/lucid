# lucid
Universal terminal interface for containers.


## Plans

* docker support
* podman support
* k8s & OpenShift support


## Status

Pre-pre-pre-alpha, please study the source code before trying it out.


## Usage

```
$ nvim -u ./vimrc
```

And in neovim type:

```
:UpdateRemotePlugins
:call _cui_init()
```
